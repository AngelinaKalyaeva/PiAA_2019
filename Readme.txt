Это репозиторий группы 7304 для лабораторных работ по дисциплине ПиАА.
У студентов нет прав коммитить в мастер, поэтому все лабораторные работы принимаются в виде pull request-ов.
Чтобы сдать лабораторную необходимо:
1) Пройти степик
2) Сделать pull request
3) Сдать лабораторную работу преподавателю на занятии

При создании pull request-ов неододимо учесть:
1) У каждого студента должны быть папка с фамилией и инициалами, в которой папка с номером лабораторной работы, внутри которой располагается код. Например:
PiAA_2019/Ivanov_II/lr_1/main.cpp
2) Убедитесь, что pull request делается из вашей ветки, а не ветки master с вашего репозитория.
